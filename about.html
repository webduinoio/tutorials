你有想過用「網頁」
就能玩轉「物聯網」的世界嗎 ?

2015 年，一群位於高雄的熱血開發者們，憑著對網頁技術源源不絕的熱情，實現真正用 Web 串接 Arduino 的技術，利用圖形化的程式編輯介面，搭配自行研發的開發版，純粹撰寫 HTML 與 Javascript 的網頁語法，就能夠玩轉各式各樣的傳感器，為廣大的開發者和網頁設計師，創造了前所未有的開發模式，並致力於降低踏入物聯網世界的門檻，堅持用最輕鬆簡單的方式，實現 IoT 開發教育。

準備好加入我們了嗎？

2017 年我們準備大步邁向全世界的程式教育及開發者市場！除了原本熱銷的「Webduino 系列開發板」，我們更準備了「雲端服務工具」蓄勢待發，為此我們將開始招兵買馬，歡迎你成為 Webduino 的一份子，一起迎接世界的舞台。

不僅有彈性的上下班時間，在法律規定的保障之外，更提供許多加碼福利，第一年就有特休、三節獎金和生日獎金、進修及旅遊補助、提供各種零食和飲料等，未來，公司更會因為你的加入而成長得更茁壯，讓你的福利加碼提升，甚至還能讓你提案要增加哪些福利！
