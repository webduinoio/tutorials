Webduino 認證中心

Webduino 認證中心提供認證課程與官方講師培訓計畫，讓對於物聯網學習與授課有興趣的你，和我們一起合作推廣程式教育。

♦ 認證中心是什麼 ♦

認證中心初衷

Webduino 提供 認證課程 與 未來官方講師培訓計畫 ，為改變大家對艱深程式學習的印象而生。

認證課程

我們備妥一系列認證課程，通過不同等級的認證考試皆代表具備不同的程式能力，您將有機會成為助理講師，並有機會加入 Webduino 官方講師群。

成為官方講師

對於官網講師資格感興趣者，除了通過 Level 1、2 所需的考試，另需參加並通過官方講師資格審定辦法，通過認證資格，將予以講師證明，期待未來能與您一同成長、茁壯，一起致力於程式教育拓展。

關於課程

LEVEL 1

本課程將實際搭配多種生活範例，引領初學者入門以外，無論是跨裝置互動控制、環境數據收集、儲存至資料庫皆不成問題，通過 Level 1 課程認證考試後，你將具備基礎的運算思維的概念，並將有資格成為 Webduino 助理講師的一員。

♦ 課程特色 ♦

  • 1. 帶走物聯網基礎開發套件組
  • 2. 12 小時精實的物聯網課程培訓
  • 3. 初步了解運算思維概念與應用 ( 變數、If else 條件判斷式... )
  • 4. 多台裝置互動
  • 5. 多元控制的豐富範例 ( 結合 Web 應用、行動裝置、Youtube 互動 )
  • 6. 課後考試認證通過將頒發證書
  • 7. 原廠授權之課程、教材、考試及午餐
  • 8. 通過 L1 考試具備官方助理講師資格

♦ 課程材料包 ♦

  • Dr. Smart 開發者套件包

♦ 考試方式 ♦

  • 通過學科考試
  • 完成術科考試
  • 評鑑上台試教狀況

LEVEL 2

本課程將延續 Level 1 運算思維概念，同步加強程式邏輯訓練，並加入更多樣化的傳感器與程式邏輯判斷。透過多種生活範例的實作，你能更精實加強程式訓練，以強化立刻動手解決生活中的問題的行動力，通過 Level 2 認證考試將具備一定程式能力，同時擁有報考官方講師之資格。

♦ 課程特色 ♦

  • 1. 帶走物聯網進階開發套件組
  • 2. 12 小時精實的物聯網課程培訓
  • 3. 活用程式邏輯概念與深入應用 ( 重複、迴圈 For、while、do while 及 Function 等概念 )
  • 4. 深入學習綜合型範例 ( 結合雲端資料庫、語音聲控、朗讀、多台裝置互動 )
  • 5. 大數據蒐集、將資料儲存於雲端資料庫
  • 6. 極完整程式教學懶人包
  • 7. 課後考試認證通過將頒發證書
  • 8. 原廠授權之課程、教材、考試及午餐
  • 9. 通過 L2 考試具備報考官方講師資格

♦ 課程材料包 ♦

  • 智慧植栽材料包
  • 智慧插座擴充套件

♦ 考試方式 ♦

  • 通過學科考試
  • 完成術科考試
  • 評鑑上台試教狀況

官方講師

自 2015 年迄今，我們經歷數百場課程與活動，倚靠著堅強的官方講師陣容，他們背負著 Webduino 教育推廣的重要使命，踏遍台灣各地進行深入教學，實現 IoT 開發教育。現今物聯網的教學需求與日俱增，我們也將擴大招募官方講師，期待未來能與我們一同成長、茁壯。

關於認證中心

Webduino 認證中心由經驗豐富的 Webduino 官方講師之團隊組成，將負責各項認證課程與程式教育推廣，從 Level 1 開始圖像化的程式語言學習，進一步於 Level 2 帶到更加深入的程式邏輯概念，最後落實各項 Web Coding Program，這種平易近人的互動式學習模式，有助於程式初學者一步步向上學習。
