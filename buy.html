Webduino 開發板

Webduino Bit

Webduino Bit 是 2018 年最新的開發板，除了原本的功能一應俱全 ( Wi-Fi 操控、多裝置串連、協同作業...等 )，並內建更多新的元件與傳感器，是目前市面上最高效能、穩定及通用的產品之一。

Webduino Smart ( 雲端開發板 )

Webduino Smart 是 2016 年推出的物聯網開發板，同時具備連上網際網路和 WebSocket 操控的能力。

Webduino 馬克 1 號

經過千錘百鍊的電路設計，Webduino 馬克 1 號終於以一身紅的姿態出現，對於手邊已經有傳感器或電子零件的開發者，可以直接藉由馬克 1 號感受 Webduino 的方便和美好。

Webduino Fly ( UNO 雲端擴充板 )

當 Arduino UNO 套用 Webduino Fly 之後，立即就變成具備透過 Wifi 控制的能力，立即擁有完整的 Webduino 開發功能，只需用 Web 技術，就可以控制 Arduino 相容的感應模組。

Webduino Fly + Arduino UNO

Arduino UNO + Webduino Fly 是目前市面上最方便的物聯網解決方案，透過 Webduino Fly 的輔助，你可以自行燒錄對應的韌體，讓開發板具備獨一無二的功能，展現您與眾不同的創造力！

Webduino 開發者套件 - 基本包

搭配 Webduino 馬克 1 號的開發套件，可以實作許多電子電路基礎範例，以及開發超音波傳感器的有趣應用。

Webduino 開發者套件 - 威力加強包

Webduino 開發者套件 - Dr. Smart

Dr. Smart 套件組內含 Webduino Smart 開發板與多個電子元件、傳感器，是 Webduino 系列產品中最精簡的一個組合。

Webduino 開發者套件 - 實戰智慧插座

內含三塊 Webduino 開發板、智慧插座與擴充套件，擁有它，不僅學會智慧插座基本控制，還能搭配網頁前後端，立即掌握智慧插座完整的開發歷程，實現做中學，學中樂的真諦！

Webduino 主題式套件

Webduino 空氣偵測套件

透過自己打造的物聯網空氣偵測器，搭配 Webduino 雲端平台，立即掌握空氣品質資料，當品質差時，自動發出警示聲，提醒您出門記得多加強防護！

Webduino 智慧家庭套件

內含 10 種傳感器與新元件，搭配 Webduino 雲端平台，一手掌握「智慧小夜燈」、「自動開關門」、「智慧門鈴」、「門禁系統」、「MP3 播放器」等多種有趣範例，迎向智慧生活。

Webduino 智慧植栽套件

Webduino 智慧植栽套件是專門針對植物栽培的套件，，內含繼電器、水泵、光敏電阻與土壤濕度偵測...等植栽專用零件組合。

Webduino 智慧插座套件 ( 材料包 )

Webduino 智慧插座擴充套件

Webduino 自走車循跡升級套件

Webduino 電子材料包

Webduino 基礎套件

入門的零件材料包，可從中學習到基本的電子電路，從點亮一顆 LED 燈開始，到自由操控七彩霓虹燈，從玩轉按鈕開關到用超音波傳感器控制 Youtube 影音播放、甚至可以偵測溫濕度來開啟風扇等。

Webduino 進階套件

內含繼電器、雙馬達驅動板以及可變電阻，可以簡單改造許多家用電路的控制，也可以輕鬆把百元玩具車改造成無線網路遙控車，還能做出像是溫度警報器、自動澆花器等有趣的監控系統。

Webduino 高階套件

針對深入研究的玩家所準備，透過這組套件，我們可以簡單實現門禁刷卡系統、輕鬆做出萬用遙控器，還可以快速做出有趣的體感遙控器等有趣的應用！

Webduino 豪華套件包

Webduino 豪華套件包裏頭包含了 13 個電子零件模組以及 7 種傳感器，當您擁有了豪華套件包，就代表您已經具備了跨足物聯網開發的進階功力！
