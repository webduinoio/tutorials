Webduino 公仔自走車設計大賞 活動辦法

自從 Webduino 公仔自走車推出以來，一直都用四種可愛的造型闖蕩江湖，今天將由您發揮巧思，畫出最特別的公仔造型，和我們一起大膽設計玩創意！自己的公仔自己創作，把 Webduino 循跡自走車帶回家！

除了這個有趣的活動，即日起所有商品全面優惠，趕快加入 Webduino 翻轉世界的行列吧！

活動時間

即日起至 4/21 (五)

活動步驟

活動期間內完成以下步驟，就有機會 帶走價值 NT$ 3,150 的循跡自走車！
還有最新的 Webduino Smart 要免費送給您！


  • STEP 1、 下載並印出空白 A4 紙公仔 ( 點選下載紙公仔檔案 )
  • STEP 2、 繪製紙公仔外觀，並將成果拍照
  • STEP 3、 留言上傳成果照片至 Facebook 活動貼文下方 ( 前往 Facebook 活動貼文 )
  • STEP 4、 公開分享 Facebook 活動貼文至個人塗鴉牆

活動好禮

  • 最佳創意獎 ( 1 名 )
    發揮您的絕妙創意，畫出最特別的公仔外觀！
    官方將以照片清晰度、創意度、完成度評選，贈送 Webduino 公仔循跡自走車 1 組 。
    ( 包含馬克一號 + 公仔自走車 + 循跡升級套件 )
  • 最佳人氣獎 ( 1 名 )
    邀請親朋好友對您的照片按讚，4/25 中午 12:00 前獲得最多按讚數的參加者 ( 至少 5 個讚以上 ) ，贈送 Webduino 公仔循跡自走車 1 組。
    ( 包含馬克一號 + 公仔自走車 + 循跡升級套件 )
  • 最佳好運獎 ( 1 名 )
    只要完成活動步驟，就可以參加抽獎！贈送 Webduino 公仔自走車 1 組。
    ( 包含馬克一號 + 公仔自走車 )
  • 熱血參與獎 ( 2 ~ 5 名 )
    不論有沒有繪製公仔造型，只要公開分享 Facebook 活動貼文至個人塗鴉牆，寫上「Webduino 循跡自走車好好玩！」，並於貼文內 tag 一位朋友，就可參加抽獎，贈送 Webduino Smart 開發板 1 片。
    ( 分享次數 50 人以內抽 2 名，50~99 人加抽 1 名，依此類推，最多抽出 5 名 )
